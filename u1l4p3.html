مثال 3

جد ناتج:
\(-3\frac{1}{2} + 2\frac{5}{6}\)
الطريقة 1: أحول الأعداد الكسرية إلى كسور غير فعلية ثم أجمعها

1. أحول العدد الكسري إلى كسر غير فعلي:

\(-3\frac{1}{2} = -\frac{7}{2}\)

\(2\frac{5}{6} = \frac{17}{6}\)

2. أجد المضاعف المشترك الأصغر (م.م.أ) للمقامين وهو 6:

\(-\frac{7}{2} = -\frac{7 \times 3}{2 \times 3} = -\frac{21}{6}\)

\(\frac{17}{6} = \frac{17}{6}\)

3. أجمع الكسور:

\(-\frac{21}{6} + \frac{17}{6} = \frac{-21 + 17}{6} = -\frac{4}{6}\)

4. أبسط الناتج:

\(-\frac{4}{6} = -\frac{2}{3}\)

الطريقة 2: أجمع الأعداد الكلية والكسور بشكل منفصل

1. أجمع الأعداد الكلية والكسور بشكل منفصل:

الأعداد الكلية: \(-3 + 2 = -1\)

الكسور: \(-\frac{1}{2} + \frac{5}{6}\)

2. أجد المضاعف المشترك الأصغر (م.م.أ) للكسور وهو 6:

\(-\frac{1}{2} = -\frac{3}{6}\)

\(\frac{5}{6} = \frac{5}{6}\)

3. أجمع الكسور:

\(-\frac{3}{6} + \frac{5}{6} = \frac{2}{6} = \frac{1}{3}\)

4. أجمع الناتجين:

\(-1 + \frac{1}{3} = -\frac{3}{3} + \frac{1}{3} = -\frac{2}{3}\)

1
جد ناتج:
\(-1\frac{1}{9} - 3\frac{1}{6}\)

تلميح: حول الأعداد الكسرية إلى كسور غير فعلية:

\(-1\frac{1}{9} = -\frac{10}{9}\)

\(-3\frac{1}{6} = -\frac{19}{6}\)

م.م.أ للعددين 9 و 6 هو 18

\(-\frac{10}{9} = -\frac{20}{18}\)

\(-\frac{19}{6} = -\frac{57}{18}\)

\(-\frac{20}{18} - \frac{57}{18} = -\frac{77}{18} = -4\frac{5}{18}\)

2
جد ناتج:
\(-2\frac{1}{3} + 4\frac{5}{12}\)

تلميح: حول الأعداد الكسرية إلى كسور غير فعلية:

\(-2\frac{1}{3} = -\frac{7}{3}\)

\(4\frac{5}{12} = \frac{53}{12}\)

م.م.أ للعددين 3 و 12 هو 12

\(-\frac{7}{3} = -\frac{28}{12}\)

\(\frac{53}{12} = \frac{53}{12}\)

\(-\frac{28}{12} + \frac{53}{12} = \frac{25}{12} = 2\frac{1}{12}\)

3
جد ناتج:
\(5\frac{1}{2} + 3\frac{1}{4} + (-\frac{11}{2})\)

تلميح: لاحظ أن \(5\frac{1}{2} = \frac{11}{2}\)، لذا:

\(\frac{11}{2} + (-\frac{11}{2}) + 3\frac{1}{4} = 0 + 3\frac{1}{4}\)

\(3\frac{1}{4} = \frac{13}{4} = 3.25\)
